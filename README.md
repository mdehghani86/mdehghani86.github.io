# Dr. Mohammad Dehghani - Faculty Website

Personal academic website for Dr. Mohammad Dehghani, Northeastern University.

## Folder Structure

```
/
├── index.html              # Main website
├── papers/                 # Publication PDFs
├── slides/                 # Presentation slides (PDF/PPTX)
├── code/                   # Code files and scripts
└── assets/
    ├── headshots/
    │   ├── faculty/        # Your headshots
    │   └── students/       # Student photos
    └── images/             # Other website images
```

## How to Add a New Paper

1. Add your PDF to `/papers/` folder
2. Copy `_TEMPLATE_paper-info.txt` and fill in details
3. Add slides to `/slides/` if available
4. Add code to `/code/` if available
5. Update `index.html` with new publication entry

## File Naming Convention

- Papers: `lastname-year-short-title.pdf` (e.g., `dehghani-2024-agentic-ai.pdf`)
- Slides: `lastname-year-short-title-slides.pdf`
- Code: `lastname-year-short-title/` (folder) or `.py`/`.txt` file
