================================================================================
HEADSHOTS FOLDER
================================================================================

This folder contains profile photos for the website.

SUBFOLDERS:
-----------
/faculty    - Your professional headshot(s)
/students   - Current and past students' photos

NAMING CONVENTION:
------------------
- Use lowercase with hyphens: firstname-lastname.jpg
- Examples:
  - mohammad-dehghani.jpg
  - john-smith.jpg

RECOMMENDED:
------------
- Square images (400x400 px or similar)
- JPG or PNG format
- Professional quality
- Consistent background if possible
